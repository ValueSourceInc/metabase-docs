## Description

按SKU累计已售经营ROI趋势（全摊薄版）。基于 #928 全摊薄已售经营表现底表，按月累计调整后净利润与全摊薄总成本，计算累计 ROI = 累计调整后净利润 / 累计总成本。 与 #910 的区别：#910 为整店口径（基于 #904，用 net_profit_usd）；本卡为按 SKU 摊薄口径（基于 #928，用 fully_loaded_net_profit_usd，已含平台费分摊）。total_cost = product_sales_usd - fully_loaded_net_profit_usd。 使用注意：筛选单个 SKU 时看到的是该 SKU 摊薄后的累计 ROI 趋势；不建议用排除某些 SKU 的方式看剩余整体（被排除 SKU 摊到的平台费会一起被排除）。
