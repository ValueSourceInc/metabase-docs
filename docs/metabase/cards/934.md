## Description

按SKU月度成本占比趋势 (area 图)。结构与 #907 完全一致：按 date:Month + cost_category 分组，sum cost_usd。 与 #907 的区别：#907 基于 #906 (含 sku=NULL 的平台级成本)，按 SKU 筛选会丢失 6 类平台成本；本卡基于 #933 (按SKU已分摊成本结构底表)，平台费已分摊到 SKU，按 SKU 筛选时各类成本完整保留，占比不失真。 筛选：通过 dashboard #70 的 SKU/Store/Marketplace/Date 筛选器映射到对应字段。与 #929/#932 一致，不带 SKU 排除 filter (by-SKU 卡靠正向选 SKU)。 已知限制：继承 #928 的 Storage 分摊约 3-4% 丢失 (某 SKU 累计销售额为 0 时无法摊到天)，其余 11 类与 #906 完全守恒。
