## Description

本表用于查看 Amazon Settlement 按 category / sub_category 汇总后的收入、费用和其他结算项。金额字段使用 USD 口径；正数通常表示收入、补偿或贷项，负数通常表示费用、退款、扣款或税金代扣。category 为空的行是按 sub_category 汇总的小计。 Subcategory 中文对照： | sub_category | 中文含义 | 说明 | |---|---|---| | product_sales | 商品销售额 | 买家支付的商品销售收入，不含后续费用扣减。 | | product_sales_tax | 商品销售税 | 商品销售相关税费，通常与 marketplace_withheld_tax 对冲，不建议纳入可控费用。 | | shipping_credits | 运费收入/配送补贴 | 买家支付或平台结算给卖家的 shipping credit。 | | shipping_credits_tax | 运费税 | shipping credit 对应税费，不建议纳入可控费用。 | | giftwrap_credits | 礼品包装收入 | 买家支付或平台结算的 gift wrap credit。 | | giftwrap_credits_tax | 礼品包装税 | gift wrap credit 对应税费，不建议纳入可控费用。 | | promotional_rebates | 促销折扣/促销返利 | 促销、coupon、折扣等产生的销售让利；在 revenue 下通常体现为收入抵减，在 cost/other 下可能体现为促销相关扣款或调整。 | | selling_fees | 销售佣金 | Amazon referral fee 等按销售收取的平台销售佣金。 | | fba_fees | FBA 履约费用 | FBA pick/pack/ship、重量尺寸相关履约费用等。 | | advertising_cost | 广告费用 | Amazon Ads 广告扣费；如做 SKU 分析，应优先使用已分摊到 SKU 的广告成本口径。 | | storage_fees | 仓储费 | FBA 月度仓储费、长期仓储或库存占用相关费用。 | | inbound_placement_fees | 入仓配置服务费 | FBA Inbound Placement Service Fee，入仓分仓/配置相关费用。 | | global_inbound_transportation_fees | 全球入仓运输费 | Amazon Global Logistics / AGS Global Inbound Transportation 等跨境入仓运输相关费用。 | | promotion_deal_fees | 促销活动服务费 | Lightning Deal、7-Day Deal、Coupon/Deal 等活动报名或服务费用。 | | removal_disposal_fees | 移除/弃置费用 | FBA 库存 removal order、disposal order 等移除或弃置费用。 | | vine_program_fees | Vine 项目费用 | Amazon Vine 测评项目相关费用。 | | subscription_fee | 月租/订阅费 | Professional Selling Plan 等店铺订阅费用。 | | reimbursement | 平台补偿/赔付 | Amazon 对丢失、损坏、调整等事项给卖家的补偿或冲销。 | | marketplace_withheld_tax | 平台代扣税 | Marketplace Facilitator Tax 等平台代收代缴税，不属于可控经营费用。 | | charity | 慈善捐赠/公益扣款 | AmazonSmile 或类似 charity 相关扣款，金额通常较小。 | | other_transaction_fees | 其他交易费用 | 未归入更细分类的交易费用或结算调整；如金额较大，应继续按 amount-type / amount-description 拆分。 | | uncategorized | 未分类 | 当前 mapping 未匹配到的项目，需要结合 amount-type / amount-description 继续检查和补充规则。 |
