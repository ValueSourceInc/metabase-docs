## Description

预测日销表：基于 wps_sales_forecast 的 start_date 和 end_date，将每条预测记录展开为每日一行，用于后续与库存表按 SKU、店铺、市场和日期进行 Join，计算未来区间销量、WOS、预计库存和补货数量。 当同一个 SKU、store、marketplace 在同一天存在多条重叠预测时，只保留优先级最高的一条记录；如果优先级相同，则保留 id 最新的一条。
